# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-01-10

## [0.1.0] - 2026-01-08

### Added

- Issue tracking with JSON-based storage in `.issues/issues.json`
- Commands: `list`, `show`, `new`, `done`, `reopen`, `note`, `update`
- ID prefix matching for convenient issue references
- Labels and criteria support for issue organization
